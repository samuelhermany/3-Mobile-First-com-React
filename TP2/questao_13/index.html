1. O que significa a abordagem Mobile-first em design web?

  • A) Desenvolver para desktops e depois adaptar para dispositivos móveis.
  • B) Criar primeiro a versão para tablets e depois para smartphones.
  • C) Desenvolver a versão para dispositivos móveis primeiro e depois adaptar para desktops.
  • D) Criar uma versão única que funcione bem em todos os dispositivos sem adaptações.

Alternativa Correta:C

A abordagem Mobile-first prioriza o desenvolvimento para dispositivos móveis, garantindo que a versão móvel seja otimizada e depois adaptando-a para telas maiores, garantindo uma experiência consistente em todos os dispositivos.

2. Qual é a principal vantagem da abordagem Mobile-first?

  • A) Menor tempo de carregamento em dispositivos móveis.
  • B) Melhor desempenho em dispositivos de desktop.
  • C) Maior flexibilidade na criação de layouts.
  • D) Facilidade em criar animações complexas.

Alternativa Correta:A

Ao focar primeiro nos dispositivos móveis, você otimiza o tempo de carregamento e melhora a performance em dispositivos com recursos mais limitados.

3. O que é uma media query em CSS?

  • A) Uma regra que altera o estilo baseado no tamanho da tela.
  • B) Uma ferramenta para testar a performance de uma página.
  • C) Uma técnica para criar animações CSS.
  • D) Um método para otimizar imagens para web.

Alternativa Correta:A

Media queries permitem que você aplique estilos diferentes com base em características do dispositivo, como o tamanho da tela, permitindo uma adaptação responsiva.

4. Como você pode garantir que um site é responsivo?

  • A) Utilizando apenas imagens de alta resolução.
  • B) Criando uma única versão para todos os dispositivos.
  • C) Usando unidades de medida relativas e media queries.
  • D) Implementando scripts de ajuste de layout para cada dispositivo.

Alternativa Correta:C

Unidades relativas e media queries permitem que o layout se ajuste dinamicamente ao tamanho da tela, garantindo uma boa experiência em diversos dispositivos.

5. Qual é a diferença entre `em` e `rem` em CSS?

  • A) `em` é relativo ao tamanho da fonte do elemento pai, enquanto `rem` é relativo ao tamanho da fonte do elemento raiz.
  • B) `em` é usado apenas para tamanhos de imagem, enquanto `rem` é para tamanhos de texto.
  • C) `em` e `rem` são unidades absolutas que não dependem do tamanho da tela.
  • D) `em` é para dispositivos móveis e `rem` para desktops.

Alternativa Correta:A

Enquanto `em` se baseia no tamanho da fonte do elemento pai, `rem` se baseia no tamanho da fonte do elemento raiz, facilitando o ajuste de tamanhos em um layout global.
