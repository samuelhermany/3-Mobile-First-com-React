Questão sobre Responsividade

Qual das seguintes opções descreve corretamente a função de uma media query em CSS?

  • A) Ajustar automaticamente a resolução de imagens.
  • B) Aplicar estilos diferentes com base em características do dispositivo, como largura da tela.
  • C) Substituir todo o conteúdo CSS em dispositivos móveis.
  • D) Definir um conjunto de regras CSS que não podem ser sobrescritas.
  • E) Garantir que todos os navegadores interpretem o CSS da mesma forma.

Resposta correta: B

Justificativa: A media query @media print é utilizada para definir estilos que serão aplicados exclusivamente ao conteúdo quando o documento for impresso. Isso permite que o desenvolvedor otimize a aparência e legibilidade do conteúdo em papel, removendo elementos desnecessários, ajustando cores, fontes e layouts para garantir uma impressão limpa e profissional.
