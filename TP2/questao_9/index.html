Questão sobre Responsividade

Qual das seguintes afirmações descreve corretamente o uso da media query @media print em um documento CSS?

  • A) É usada para aplicar estilos específicos para telas de dispositivos móveis.
  • B) É usada para aplicar estilos específicos para impressão.
  • C) Ativa o estilo Retrato ou paisagem na impressão.
  • D) Aplica estilos apenas para dispositivos com resolução de tela superior a 1080p.
  • E) Aplica estilos específicos apenas para navegadores móveis.

Resposta correta: B

Justificativa: A media query @media print é utilizada para definir estilos que serão aplicados exclusivamente ao conteúdo quando o documento for impresso. Isso permite que o desenvolvedor otimize a aparência e legibilidade do conteúdo em papel, removendo elementos desnecessários, ajustando cores, fontes e layouts para garantir uma impressão limpa e profissional.
